## 课程概览

### 第一部分：C++面向对象程序设计（实验1-8）

| 实验 | 主题 | 内容 |
|------|------|------|
| 实验1 | C++基础 | 循环、函数、温度转换 |
| 实验2 | 函数进阶 | 最大值/最小值函数、ASCII字符处理 |
| 实验3 | 循环与数学 | 水仙花数、求根公式、Pi级数 |
| 实验4 | 循环应用 | 百钱买百鸡、指针对应、冒泡排序 |
| 实验6 | 类与运算符重载 | sin/cos泰勒展开、复数类、凯撒密码 |
| 实验7 | 类、继承、引用 | 引用参数、圆柱类、有理数类 |
| 实验8 | 多态与继承 | 形状类层次（Circle/Square/Rectangle）、教授继承、多继承 |

### 第二部分：数据结构（week2-17）

| 周次 | 主题 | 关键内容 |
|------|------|----------|
| week2 | 入门 | 七桥问题（欧拉回路）、荷兰国旗问题 |
| week3 | 顺序表 | 顺序表类的插入、删除、查找操作 |
| week4 | 单链表 | 单链表实现、学生信息管理 |
| week5 | 链表应用 | 一元多项式求和、约瑟夫环问题 |
| week7 | 栈和队列 | 顺序栈、链栈、循环队列、括号匹配、表达式求值 |
| week8 | 串与KMP | KMP模式匹配、表达式计算 |
| week9 | 哈夫曼编码 | 哈夫曼树构造、稀疏矩阵 |
| week10 | 二叉树 | 前/中/后序遍历（非递归实现）、由前序中序创建二叉树 |
| week12 | 图的遍历 | 邻接矩阵/邻接表的遍历 |
| week13 | 图的表示 | 邻接矩阵类 MGraph、邻接表类 ALGraph |
| week14 | 图算法 | Dijkstra、Floyd、Prim、Kruskal算法 |
| week15 | 拓扑排序 | 有向无环图（DAG）的拓扑排序 |
| week16 | 查找与排序 | 二叉排序树、各种内部排序算法（插入/希尔/冒泡/快速/选择/堆/归并） |
| week17 | 综合 | 期末综合练习 |

## 排序算法汇总（week16）

- 直接插入排序 (Insert Sort)
- 希尔排序 (Shell Sort)
- 冒泡排序 (Bubble Sort)
- 快速排序 (Quick Sort)
- 选择排序 (Select Sort)
- 堆排序 (Heap Sort)
- 归并排序 (Merge Sort) — 递归与迭代

## 图算法汇总（week14）

- **最短路径**：Dijkstra算法、Floyd算法
- **最小生成树**：Prim算法、Kruskal算法
